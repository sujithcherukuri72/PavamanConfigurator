## ?? Color Coding

### **Severity Badges**

| Severity | Background | Text Color | Use Case |
|----------|------------|------------|----------|
| **Info** | `#EEF2FF` (Light Blue) | `#3B82F6` (Blue) | Normal events, mode changes |
| **Warning** | `#FEF3C7` (Light Yellow) | `#F59E0B` (Orange) | GPS poor, high vibration |
| **Error** | `#FEE2E2` (Light Red) | `#EF4444` (Red) | Battery low, sensor errors |
| **Critical** | `#FEE2E2` (Light Red) | `#DC2626` (Dark Red) | Crash, critical battery |

### **Graph Field Colors**

Fields are assigned colors in sequence:

1. `#3B82F6` - Blue (AHR2.Lat)
2. `#EF4444` - Red (AHR2.Lng)
3. `#10B981` - Green (AHR2.Alt)
4. `#F59E0B` - Orange (next field)
5. `#8B5CF6` - Purple
6. `#EC4899` - Pink
7. `#06B6D4` - Cyan
8. `#84CC16` - Lime
9. `#F97316` - Orange-Dark
10. `#6366F1` - Indigo

## ?? Interactive Elements

### **Plot Tab**

| Element | Interaction | Result |
|---------|-------------|--------|
| Field checkbox | Click | Toggle field on/off graph |
| Field name | Click | Same as checkbox |
| Search box | Type | Filter fields by name |
| Color square | Visual | Shows assigned color |
| Legend entry | Hover | Show full stats |

### **Events Tab**

| Element | Interaction | Result |
|---------|-------------|--------|
| Event row | Click | Jump to time in Plot |
| Severity filter | Check/Uncheck | Show/hide severity level |
| Search box | Type | Filter events by text |
| Severity badge | Visual | Color-coded severity |

## ?? Usage Tips

### **Best Practices**

1. **Select Related Fields:**
   ```
   For attitude analysis:
   - AHR2.Roll
   - AHR2.Pitch
   - AHR2.Yaw
   ```

2. **Filter Critical Events:**
   ```
   Uncheck Info and Warning
   ? See only Error and Critical
   ? Focus on problems
   ```

3. **Compare GPS Coordinates:**
   ```
   Select both:
   - GPS.Lat
   - GPS.Lng
   ? See flight path in data
   ```

4. **Track Battery Voltage:**
   ```
   Select:
   - BAT.Volt
   - BAT.Curr
   ? Monitor power consumption
   ```

## ?? What to Look For When Testing

### **Checkbox Behavior**

? **CORRECT:**
- Click field ? Checkbox instantly checks (?)
- Click again ? Checkbox instantly unchecks (?)
- Color square appears/disappears with selection
- Graph updates when selection changes

? **WRONG (if still broken):**
- Click field ? Checkbox stays unchecked
- Need to click twice to select
- Color square doesn't match checkbox state
- Graph shows field but checkbox doesn't

### **Events Display**

? **CORRECT:**
- Events list populates after log loads
- Severity badges show colored backgrounds
- Event counts display in Overview tab
- Filters work instantly

? **WRONG (if still broken):**
- Events tab stays empty
- No severity badges (plain text)
- Event summary shows all zeros
- Filters don't work
